阵列 ​

↓abcR1cbaR2bcc

markdown 公式如下：

markdown
$$
\begin{array}
{c|ccc}
{↓}&{a}&{b}&{c}\\
\hline {R_1}&{c}&{b}&{a}\\
{R_2}&{b}&{c}&{c}\\
\end{array}
$$
  • 需要 array 环境：起始、结束处以{array}声明
  • 对齐方式：在{array}后以
    • 左对齐：l ；剧中：c；右对齐：r
    • 竖直线：在声明对齐方式时，插入|建立竖直线

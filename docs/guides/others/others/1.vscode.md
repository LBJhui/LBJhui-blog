## vscode 创建代码模版

1. 在 VSCode 中，按下 Ctrl + Shift + P（Windows/Linux）或 Cmd + Shift + P（Mac）打开命令面板。
2. 然后输入"Preferences: Configure User Snippets"并选择该选项。打开一个 json 文件用户创建模版，也可以新创建一个 json 文件
